# Authoring with Fluid
Focus so far has been on UX (user experience) What's needed from authoring point of view? * Command-line publishing tool -- currently need to install PureScript and write a small program! * Fluid kernel for Jupyter * Authoring with Quarto, Markdown * GitHub Actions?
What about the language itself? * Purely functional (no side-effects) * Records, lists, dictionaries, 2D arrays * Graphics library based on d3.js What's missing? * Modules/imports * Data sources (load from file/URL/DB) * Syntax for datatype definitions
